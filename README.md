
## Description

Sample Api + Nest.js + Graphql 

## Installation

```bash
$ yarn install
$ copy .env.example .env
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
